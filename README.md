# Milo's Dark Pattern Run

This is a browser-based HCI project starter built with HTML, CSS, and JavaScript.

## Included assignment requirements

- Welcome / onboarding screen
- 10 task gameplay loop
- Visible score and timer
- Immediate feedback after each action
- Escalation moment with higher pressure and shorter timers
- Final score screen
- Replay option
- Debrief screen explaining each dark pattern used

## Main idea

The player controls a small character named Milo who moves through a fake app city.
Each challenge asks the player to resist a different dark pattern.

## Files

- `index.html`: screen structure
- `style.css`: visuals and layout
- `script.js`: tasks, scoring, escalation, and debrief logic
- `starter.html`: simpler backup sample
